Summary

There are four different model types currently "fitable"

  1. Timeexpansion No, Mixed No : fit(UnfoldLinearModel,f,evts,data_epoch,times)
  2. Timeexpansion Yes, Mixed No : fit(UnfoldLinearModel,f,evts,data,basisfunction)
  3. Timeexpansion No, Mixed Yes : fit(UnfoldLinearMixedModel,f,evts,data_epoch,times)
  4. Timeexpansion Yes, Mixed Yes: fit(UnfoldLinearMixedModel,f,evts,data,basisfunction)
